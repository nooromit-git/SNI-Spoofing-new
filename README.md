# SNI-Spoofing-new

🚀 **نسخه جدید SNI: دسترسی سریع و پایدار به اینترنت آزاد با کانفیگ Vless**

این پروژه یک راهکار پیشرفته و بهینه‌شده برای دور زدن محدودیت‌های شدید اینترنتی با استفاده از تکنیک اسنیفینگ و تغییر SNI در پروتکل‌های امن است. با ترکیب این متد با کانفیگ‌های قدرتمند **Vless**، می‌توانید به بالاترین سرعت ممکن، پایداری فوق‌العاده و کمترین پینگ دسترسی داشته باشید.

---

## 📌 ویژگی‌های کلیدی (Features)

- ⚡ **سرعت و پایداری بالا:** بهینه‌سازی شده برای شبکه داخلی و بین‌المللی جهت کاهش پکت لاس (Packet Loss).
- 🔒 **تغییر هوشمند SNI:** عبور از فیلترینگ لایه ۷ با بهره‌گیری از هدرهای معتبر.
- 📱 **سازگاری کامل:** قابل استفاده در تمامی نرم‌افزارهای کلاینت V2Ray (مانند v2rayNG، Nekobox، v2rayN، Sing-box و Shadowrocket).
- 🛠️ **پیکربندی آسان:** بدون پیچیدگی‌های فنی اضافه، آماده برای استفاده سریع.

---

## ⚠️ نکته بسیار مهم (⚠️ VERY IMPORTANT)

> **توجه:** این مخزن شامل اسکریپت‌ها و ابزارهای تنظیمات است. شما برای اتصال **حتماً باید کانفیگ‌های Vless بهینه‌شده را از داخل فایل `configs.txt` موجود در همین پروژه کپی کنید** و داخل نرم‌افزار کلاینت خود وارد (Import) نمایید. از کپی کردن مستقیم متون متفرقه خودداری کنید تا اختلالی در SNI ایجاد نشود.

## 🛠️ نحوه استفاده و راه‌اندازی (How to Use)

1. **دریافت فایل‌ها:** ابتدا کل این پروژه را دانلود یا کلون کنید:
   ```bash
   git clone [https://github.com/your-username/SNI-Spoofing-new.git](https://github.com/your-username/SNI-Spoofing-new.git)
کپی کانفیگ‌ها: فایل configs.txt را باز کرده، تمام متن داخل آن یا کانفیگ مورد نظر خود را کپی کنید.

وارد کردن به کلاینت: نرم‌افزار خود (مثلاً v2rayNG یا Nekobox) را باز کرده و گزینه‌ی Import config from Clipboard را بزنید.

تست و اتصال: کانفیگ را انتخاب کرده، تست پینگ بگیرید و متصل شوید.
